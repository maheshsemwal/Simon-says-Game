Instructions

  • Press any key to start the game
  • Simon will give the first signal
  • Repeat the signal by pressing the same color button
  • Simon will duplicate the first signal and add one
  • Repeat these two signals by pressing the same color buttons, in order
  • Continue playing as long as you can repeat each sequence of signals correctly
  • If you fail to repeat a sequence exactly, Simon responds with a "RAZZ" sound. This means you've lost, and the sequence of signals ends
  • If you manage to repeat of the first 13 signals, you have won the game
